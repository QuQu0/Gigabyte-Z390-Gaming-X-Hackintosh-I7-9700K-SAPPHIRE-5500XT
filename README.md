## 修改BIOS

需要解锁cfg lock，直接修改BIOS重新刷入，一劳永逸解决问题。   
修改版BIOS需要在DOS命令行下刷入，不能使用官方工具刷入。  
也可以使用命令行方式关闭cfg lock 
此外在声音滑块在滑动中，存在飘动卡顿现象，  
此故障需要关闭BIOS中的serial port功能修复，但是BIOS设置中已经取消，需要修改BIOS重新刷入。
